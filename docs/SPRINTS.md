# Sprints

Ce document enregistre les sprints des deux tracks:
- Track 1: Mobile compatible
- Track 2: Esthetique sobre premium

## Track 1 — Mobile compatible (passe complete)

### Sprint M1 — Layout & navigation mobile
- Breakpoints Android (min 360px)
- Navigation mobile (menu compact, header sticky)
- Home calendrier en version mobile
- CockpitWeek : table → cartes empilees
- Modals full-screen avec actions visibles

### Sprint M2 — Interactions & lisibilite
- Zones tactiles >= 44px
- Densite/espacement/tailles de texte
- Scrolls propres (sections, modals, chat)
- Etats actifs clairs

### Sprint M3 — Stabilisation mobile
- QA mobile sur toutes les pages
- Fix overflow / focus / inputs
- Ajustements performance visuelle

## Track 2 — Esthetique sobre premium (passe complete)

### Sprint UI1 — Systeme visuel
- Palette ivoire/beige + gris bleute
- Accent discret (1 couleur)
- Nouvelle typographie
- Variables CSS globales
- UI kit leger (boutons, cards, inputs, badges)

### Sprint UI2 — Application pages
- Home (calendrier, header, actions)
- CockpitWeek (listes, chat, actions)
- Modals (menus/recettes/courses)

### Sprint UI3 — Finitions
- Harmonisation globale
- Micro-animations utiles
- Etats de chargement/erreur sobres
